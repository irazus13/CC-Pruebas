## Resumen

Dental implant failure is influenced by anatomical, systemic, and procedural factors. This study assessed the predictive value of machine learning models: logistic regression, Random Forest, and gradient boosting, using an open-access dataset (Liu et&#xa0;al. 2018) containing demographic, surgical, prosthetic, and systemic variables. Random Forest performed best (accuracy 0.85, ROC-AUC 0.79, F1&#x2009;=&#x2009;0.92, recall 0.97), followed by gradient boosting, while logistic regression showed lower sensitivity. Feature importance analysis identified implant location, sinus augmentation, implant dimensions, and patient age as key predictors. Ensemble models and interpretable feature metrics demonstrate strong potential for improving clinical risk stratification in implant dentistry.
